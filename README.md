# General Tree Application

This project is a Java console-based application that demonstrates a **general tree implementation using arrays**. The program includes several test operations such as:

- Finding descendants (direct and all)
- Finding siblings of a species
- Finding the ancestor of a species

The project can be run either using the **JAR file in the terminal** or using the **Windows executable (.exe)**.

---

## Requirements

To run this project using the JAR file, you must have:

- **Java JDK or JRE 17+ installed**
- Java added to your system `PATH`

You can verify your Java installation using:

java --version
